# МРЕО Монитор — Telegram Бот (Playwright версия)

Бот следит за доступными номерными знаками на сайте ГСЦ МВС Украины
через настоящий браузер.

## Файлы
- `bot.py` — основной код
- `Dockerfile` — конфигурация для Railway
- `requirements.txt` — Python зависимости
- `.dockerignore` — что не включать в образ

## Переменные окружения на Railway
- `BOT_TOKEN` — токен от BotFather
- `ADMIN_ID` — ваш Telegram ID

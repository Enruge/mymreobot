# МРЕО Монитор — Telegram Бот

Бот следит за доступными номерными знаками на сайте ГСЦ МВС Украины
(Харьковская область) и уведомляет в Telegram раз в день.

## Команды бота

| Команда | Что делает |
|---------|-----------|
| /list | Показать список отслеживаемых комбинаций |
| /add 1234 | Добавить комбинацию |
| /remove 1234 | Удалить комбинацию |
| /check | Проверить прямо сейчас |
| /help | Справка |

## Деплой на Railway

### 1. Загрузить файлы на GitHub
1. Зайдите на github.com → New repository → назовите `mreo-bot`
2. Загрузите все файлы из этой папки

### 2. Задеплоить на Railway
1. Зайдите на railway.app → New Project → Deploy from GitHub repo
2. Выберите репозиторий `mreo-bot`
3. Перейдите в Variables и добавьте:
   - BOT_TOKEN = (токен от @BotFather)
   - CHAT_ID = (ваш ID от @userinfobot)
4. Нажмите Deploy

## Переменные окружения

| Переменная | Описание |
|-----------|---------|
| BOT_TOKEN | Токен бота от @BotFather |
| CHAT_ID | Ваш числовой ID в Telegram |
